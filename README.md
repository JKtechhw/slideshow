# Slideshow

## Dependences
- NPM - express, nodemon, mongodb
- NODE.js
- MONGODB

## Install dependences
Debian based
```markdown
sudo apt install nodejs npm
```
Arch based
```markdown
yay -S nodejs npm
```


## Instalation
In cloned repository directory run:
```markdown
npm install
```
